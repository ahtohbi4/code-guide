Syntax
--

#### Always declare the document type

And do it as the first line in your document.
```html
<!doctype html>
```

#### Use lower case element names

Lowercase look cleaner and are easier to write.
```html
<!-- Bad -->
<Header>
    <A class="logo" href="//example.com/"></A>
</Header>
<!-- Good -->
<header>
    <a class="logo" href="//example.com/"></a>
</header>
```
